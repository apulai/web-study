TodoList app

This is a node.js based application with Express and Mongoose. The aim is to create a ToDo list application. The backend is mongodb running on the same host, as the main application.
To test it you have to install node.js on your host. I've used an LTS version from https://nodejs.org/en/. First init the package repo with node init, then download the node.js packages with node -i, finally run the app with node app.js For mongodb I've used docker pull mongodb. Start your container with docker run -d -p: 27017:27017 mongo. Finally direct your browser to http://localhost:3000/
