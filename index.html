TodoList app

This is a node.js based application with Express and Mongoose. The aim is to create a ToDo list application. The backend is mongodb running on the same host, as the main application.
To test it you have to install node.js on your host. I've used an LTS version from https://nodejs.org/en/. First init the package repo with node init, then download the node.js packages with node -i, finally run tha app with node app.js
